# Frontend Fix Sprint Design

## Goal

Create a sellable frontend service system for Ahmed Mohsen that can realistically support an additional 500 USD/month through small, fast React/Next.js work.

## Chosen Path

Option 5: a ready-to-sell frontend bug-fixing and UI improvement service.

This is the fastest path because it sells Ahmed's existing skill directly. It does not require building a marketplace audience for templates, validating a micro SaaS idea, writing backend systems, or waiting for recurring product revenue. The target outcome is two 250 USD projects/month or one 500 USD monthly maintenance retainer.

## Offer

Frontend Fix Sprint:

- Fix 3-5 React/Next.js frontend bugs, responsiveness issues, UI polish issues, or performance problems.
- Deliver a clean pull request, before/after notes, and a short handoff checklist.
- Standard price: 250 USD/project.
- Retainer option: 500 USD/month for two monthly sprints plus priority support.

## Demo App

The demo app is a Next.js site with two jobs:

- Sell the service clearly on the home page.
- Prove the quality of work through a premium SaaS dashboard improvement demo.

The demo should feel like the kind of polished SaaS/admin interface Ahmed can deliver: dense enough for real work, responsive, accessible, and specific.

## Architecture

- Next.js App Router with TypeScript.
- Tailwind CSS for styling.
- Feature-first source structure.
- Static mock data for the dashboard and service copy.
- Small reusable UI primitives for buttons, badges, metrics, and dashboard panels.
- Vitest tests for offer pricing and fit-scoring logic.

## Pages

- `/`: service landing page with offer, outcomes, proof, pricing, FAQs, and CTA.
- `/demo`: SaaS dashboard polish demo showing realistic business metrics, issues, tasks, and before/after improvements.

## Success Criteria

- The repository includes real sales assets, workflow docs, and reusable client prompts.
- The app builds successfully.
- The service offer is clear enough to send to a founder, agency, recruiter, or Upwork buyer.
- The demo visually supports a 250 USD project price.
- Ahmed can run the app locally and deploy it to Vercel.
