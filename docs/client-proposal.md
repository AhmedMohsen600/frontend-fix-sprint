# Client Proposal

## Option A: Frontend Fix Sprint

I will fix 3-5 scoped React/Next.js frontend issues in 3 days. Good fits include UI bugs, responsiveness problems, dashboard polish, small performance improvements, broken states, spacing issues, and landing page polish.

Price: 250 USD fixed project fee.

## Option B: Frontend Cleanup Sprint

I will refactor one messy React/Next.js feature or module into cleaner, maintainable structure without changing the current UI or breaking existing behavior.

Good fits include huge components, duplicated logic, messy state, poor folder structure, mixed UI/business logic, hard-to-maintain forms, unreadable Tailwind classes, and fragile frontend features.

Price: 300-500 USD depending on scope.

Timeline: 3-5 days for one scoped feature/module.

## What I Need From The Client

- App URL or preview link.
- GitHub/repo access.
- Exact issue list or exact feature/module to clean.
- Screenshots, Loom/video, or Figma file if relevant.
- Local setup instructions and required environment variables.
- Test account if the feature is behind login.
- Clear approval contact.

## What The Client Gets

- Implemented frontend fixes or scoped refactor.
- Clean pull request or delivered source files.
- Verification notes.
- Before/after notes when visual changes matter.
- Refactor notes explaining what changed when cleanup work is included.
- Any recommended next steps outside the agreed scope.

## Payment Terms

- 50% upfront to reserve the sprint.
- 50% on delivery before final handoff or merge.
- One focused revision pass is included for items inside the agreed sprint scope.

## Delivery Process

1. Confirm offer type, scope, and acceptance criteria.
2. Receive access and reproduce/review the issue or messy module.
3. Implement the fixes or cleanup.
4. Run available checks such as lint, typecheck, tests, and build.
5. Send delivery notes and review link or pull request.
6. Apply one scope-specific revision pass if needed.
