# 7-Day Launch Plan

Ahmed's required time: 30-60 minutes/day.

## Day 1

- Run the demo locally.
- Add the demo link to your LinkedIn profile featured section if deployed.
- Send the LinkedIn post from `docs/sales-messages.md`.

## Day 2

- Send 10 founder DMs.
- Use the founder message from `docs/sales-messages.md`.
- Target people with SaaS apps, dashboards, landing pages, or public product launches.

## Day 3

- Send 10 agency DMs.
- Target small web/product agencies that build React or Next.js projects.
- Offer overflow frontend help.

## Day 4

- Create or update one Upwork profile service around "React/Next.js frontend bug fixing and UI polish."
- Send 5 tailored Upwork proposals using the provided proposal as the base.

## Day 5

- Publish one mini case study using the demo app.
- Show before/after dashboard polish, even if it is a self-initiated demo.

## Day 6

- Follow up with everyone who replied or viewed the offer.
- Send the follow-up message.
- Ask for one small issue list, URL, or screenshot.

## Day 7

- Review responses.
- Convert interested leads into either:
  - 250 USD Frontend Fix Sprint, or
  - 500 USD Monthly Frontend Care.
- Paste the first real client request into Codex using the reusable prompt.

## Weekly repeat

- 20 founder/agency DMs.
- 5 Upwork proposals.
- 1 public proof post.
- Follow up once.
